# Asset Management

To update the images on the website, simply replace the following files in this folder (`public/images/`) with your own high-resolution photos.

**Important:** Please try to keep the file names the same, or update the code references if you change them.

## Required Files

1.  **`hero.png`**: The large banner image on the homepage.
    *   *Recommended Size*: 1920x1080px or larger.
    *   *Style*: Dark, moody, horizontal orientation.

2.  **`workshop.png`**: The image used for workshop listings and the "Artisan Touch" section.
    *   *Recommended Size*: 800x600px.
    *   *Style*: Action shot of making candles or a beautiful setup of tools.

3.  **`products.png`**: The default image for product listings.
    *   *Recommended Size*: 800x800px (Square).
    *   *Style*: Clean product shot on a neutral background.

## How to replace
1.  Rename your new photo to `hero.png` (or `workshop.png`, etc).
2.  Drag and drop it into this folder, overwriting the existing file.
3.  Refresh your browser.
